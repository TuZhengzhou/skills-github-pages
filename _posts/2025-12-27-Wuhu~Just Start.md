So we follow the guidance and created a new file. We use Jekyll in GitHub Pages, and the blog must obey the file naming pattern and the "frontmatter". 
The file name must be like `_posts/YYYY-MM-DD-title.md`, and the file front must contain `title` field and `date` field.

What is **frontmatter**? The pattern used by Jekyll files are called **YAML frontmatter**, it is on top of the file, and looks like:
```
---
title: "Welcome to my blog"
date: 2019-01-20
---
```
